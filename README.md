# AUR – Asia University Ranking

AUR (Asia University Ranking) is a modern web platform that helps students, researchers, and educators discover, compare, and analyze universities across Asia through rankings, analytics, and intelligent search capabilities.

The platform provides real-time ranking insights, university exploration tools, analytics dashboards, and educational content in a responsive and user-friendly interface.

---

## Features

### University Directory
- Browse universities across Asia
- Search institutions by name or location
- View detailed university profiles
- Explore country-wise university listings

### University Rankings
- Live ranking dashboard
- Top-ranked universities in Asia
- Ranking score comparisons
- Research and employability metrics

### Analytics Dashboard
- Interactive data visualization
- Ranking trends and insights
- Academic performance metrics
- Comparative analytics

### Country Exploration
- Browse universities by country
- Regional education insights
- Country-wise ranking analysis

### Advanced Filtering
- Institution search
- Country filters
- Ranking-based filtering
- Category-based exploration

### Blog & Insights
- Higher education articles
- University highlights
- Academic trends and updates

### AI-Powered Assistance
- Intelligent search support
- Educational guidance
- Interactive user assistance

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/<username>/aur.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
